โครงการล่าสุดคือ  ได้ทำเว็บไซต์ (Freeland) ชื่อ www.ครูฟลุ๊คขอสอนสังคม.com ส่วน (ผลงานอื่นๆ  ไม่สามารถเปิดเผยได้)  ก็สามารถปิดโครงการนี้ได้ตามเป้าหมาย ปัจจุบัน เป็นทหาร ซึ่งช่วงหลังๆ พอมีเวลาว่างจึงรับงานมาทำเพื่อเป็นรายได้เสริม และเมื่อไม่มีงาน ก็ศึกษา การเขียนโค๊ดรูปแบบใหม่ๆ อย่างเช่น Laravel
ก่อนหน้านี้ งานบริษัทเกี่ยวกับพัฒนาระบบ ซึ่งหน้าที่หลักๆ คือ develop web ด้วย ภาษา php เป็นหลัก ซึ่งไม่ได้ใช้ freamwork ตัวใด แต่ทางบริษัทมีระบบมารองรับในการเขียนให้สำหรับโปรแกรมเมอร์ ซึ่งสามารถทำงานได้ง่าย และรวดเร็วในการทำงาน
ผมจึงไม่เคยใช้ freamwork ตัวใดเลย เพราะตอนเรียน ก็ไม่มีสอน ผมจึงต้องศึกษาเพื่อพัฒนาตนเอง ลองสร้างโปรเจคเล็กๆมาทำดู ก็สามารถเข้าใจได้ด้วยเวลาไม่นานนัก และปัจจุบันกำลังฝึกฝน node js + react
ปัญหาหลักๆที่พบ ในการพัฒนาตนเองคือ ไม่มีใครช่วยสอน จึงต้องค้นคว้า และหาคำตอบเอง จึงใช้เวลานานในบางปัญหา แต่ท้ายที่สุดก็สามารถแก้ปัญหานั้นได้
